A two-page interactive Power BI dashboard providing 360° insights into customer enquiries and sales behavior. The solution analyzes enquiry trends, order sources, sales group performance, and process-wise contribution using Financial Year–based logic. It includes KPI cards, donut charts, trend analysis, and detailed customer-level drill-downs. Optimized DAX measures and an efficient data model ensure high performance and scalability. Designed with a clean, business-friendly UI to support data-driven decision-making.
